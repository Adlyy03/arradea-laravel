# Notification Sounds

Place your notification sound file here as `notification.mp3`.

## Recommended Specifications

- **Format**: MP3
- **Duration**: 1-3 seconds
- **File Size**: < 100KB
- **Sample Rate**: 44.1kHz
- **Bit Rate**: 128kbps

## Free Sound Resources

You can download free notification sounds from:

1. **Zapsplat** - https://www.zapsplat.com/sound-effect-categories/notification-sounds/
2. **Freesound** - https://freesound.org/search/?q=notification
3. **Notification Sounds** - https://notificationsounds.com/
4. **Mixkit** - https://mixkit.co/free-sound-effects/notification/

## Example Sound

If you don't have a sound file, you can use a simple beep sound or download one from the resources above.

The sound file should be named: `notification.mp3`
